Newsfeed test
============

directory structure:

`src/api/` - api sourcecode

`src/utils/` - helpers to get and process news data

`src/types.ts` - all types needed for NewsFeed model 

`src/__tests__` - unit tests

TESTS
------

to test project
```
npm test
```

START 
-----
to start project:
```
npm start
curl http://localhost:3001
```
